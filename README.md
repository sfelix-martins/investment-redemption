# Investment Redemption

Uma mini aplicação que react native que simula um resgate personalizado de fundos de investimentos em ações.

## Tecnologias/Frameworks Utilizadas

- React Native
- React Hooks
- React Navigation
- React Native Paper (Material Design)
- SWR
- Axios
- React Testing Library
- Jest
- Typescript

## Uso

Após clonar o projeto, acesse a paste e:

- Instale as dependências com o comando `yarn install`
- Execute o app na plataforma desejada com `yarn ios` ou `yarn android`


## Testes

Para executar os testes automatizados utilize o comando `yarn test`p
